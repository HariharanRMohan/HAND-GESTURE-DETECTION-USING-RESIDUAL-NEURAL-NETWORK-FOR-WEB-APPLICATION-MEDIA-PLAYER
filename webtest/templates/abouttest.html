About Our Project!!!

The purpose of Hand Gesture Detection Using Residual Neural Network for Web Application Media Player project build is to advance the technology in gesture recognition. Beside using traditional ways, the world is looking for a new advance method to overtake the traditional method. From that, users will feel satisfied by using it in this technological era. Most of the people love to involve in media and entertainment activities and by integrating the project to them, the users feel new about this technology, and they can adapt into it quickly without any problem.

This project has a strong strength that used Artificial Intelligence technique that involves gesture recognition. By having gesture recognition, it makes the user easier to control the media player. While doing other things, they can control the player by using their hand only. This project gives user a new way to interact that is like how they interact to the actual world. They feel natural and require neither interruption nor an additional device. This project can be used for future upgrades as it can be used in car infotainment systems, video game consoles, and human-machine interfaces. The basic fact is that gesture recognition technology is growing popularity, with no signs of slowing down anytime soon. The use of gesture recognition in the creation of interactive and compelling live experiences and enormous promise.
